What this calculator does not include

  • Battery replacement cost — typically 8–10 year lifespan, can cost a significant portion of the HEV premium when due.
  • Resale value — HEVs currently hold value well but tech obsolescence is a real risk.
  • Maintenance differences — HEV brakes last longer (regenerative braking), but hybrid system servicing can be pricier.
  • Opportunity cost — the PKR 2.65M premium invested at 15% could outpace fuel savings on low mileage.
  • Feature differences — the two variants have different spec levels; this is pure fuel-cost math.
